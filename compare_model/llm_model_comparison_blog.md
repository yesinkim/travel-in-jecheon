# 한국어 LLM 모델 실전 테스트: 질문별 답변 상세 분석

9가지 실제 질문으로 6개의 한국어 LLM 모델을 테스트하고, 각 모델이 어떻게 답변했는지 상세히 분석했습니다.

## 테스트 모델 소개

### 7~8B 모델
- **Qwen2.5-7B** (Alibaba Cloud) - 평균 6.50초
- **EXAONE-3.5-7.8B** (LG AI Research) - 평균 7.85초
- **Kanana-1.5-8B** (Kakao) - 평균 10.10초

### 1~2B 모델

- **EXAONE-4.0-1.2B** (LG AI Research) - 평균 8.26초
- **Kanana-nano-2.1b** (Kakao) - 평균 26.83초

#### 7~8B 모델

**Qwen2.5-7B**
- ✅ 강점: 가장 빠른 속도, 간결한 답변
- ❌ 약점: 한국 문화 이해 부족 (삼국시대 혼동), 형식 요청 무시 경향

**EXAONE-3.5-7.8B**
- ✅ 강점: 한국어 특화, 정확하고 체계적, 균형잡힌 답변
- ❌ 약점: 일부 사실 오류 (제천 위치 설명에서 "충청북도가 충청북도와 통합")

**Kanana-1.5-8B**
- ✅ 강점: 가장 상세한 설명, 한국 문화 깊은 이해, 교육적
- ❌ 약점: 느린 속도, 때때로 과도하게 긴 답변

#### 1~2B 모델

**Qwen2.5-1.5B**
- ✅ 강점: 빠른 속도 (일부 질문)
- ❌ 약점: 심각한 정확도 문제, 영어/중국어 혼용, 환각 많음

**EXAONE-4.0-1.2B**
- ✅ 강점: 균형잡힌 성능, 대부분 정확한 답변
- ❌ 약점: 일부 질문에서 영어 사용, 사실 오류 (AKB48, 존재하지 않는 그룹명)

**Kanana-nano-2.1b**
- ✅ 강점: 높은 정확도, 상세한 답변, 7~8B급 품질
- ❌ 약점: 매우 느림 (약 27초 고정), 실시간 서비스 부적합

### 최종 추천

#### 프로덕션 서비스
→ **EXAONE-3.5-7.8B** (한국어 특화, 균형)

#### 빠른 응답 필요
→ **Qwen2.5-7B** (정확도 검증 필수)

#### 교육/콘텐츠 생성
→ **Kanana-1.5-8B** (가장 상세)

#### 경량 환경
→ **EXAONE-4.0-1.2B** (1~2B 중 최선)

#### 배치 처리
→ **Kanana-nano-2.1b** (느리지만 정확)
